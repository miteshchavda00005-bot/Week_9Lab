Project Description

This repository contains the Week 9 lab assignment. The purpose of this lab is to practice web development concepts including JavaScript functionality, event handling, and user interaction in a web application.

The program demonstrates how different components of a web page interact with JavaScript to perform actions based on user input. The application processes user selections and dynamically updates the interface based on those selections.
